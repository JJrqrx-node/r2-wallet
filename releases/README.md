# Release Archives

This folder holds versioned ZIP builds of the R2 Wallet extension for users who can't or don't want to use the Chrome Web Store.

- Release notes: [v0.1.0](https://github.com/JJrqrx-node/r2-wallet/releases/tag/v0.1.0)

### Install

1. Download `r2-wallet-0.1.0.zip`
2. Unzip the file locally
3. Open `chrome://extensions` and toggle Developer mode on
4. Click **Load unpacked** and select the unzipped folder
